## Proje Hakkında
Bu program, $ax^2 + bx + c = 0$ formundaki denklemlerin köklerini hesaplar.

## Özellikler ve Akış
1. *Veri Girişi:* Kullanıcıdan a, b ve c katsayılarını alır.
2. *Hesaplama:* $\Delta = b^2 - 4ac$ formülü ile diskriminantı bulur.
3. *Analiz:* - *Delta > 0:* İki farklı reel kök hesaplar.
   - *Delta = 0:* Tek bir reel kök hesaplar.
   - *Delta < 0:* Reel kök olmadığını belirtir.

## Nasıl Çalışır?
C++ derleyicisi ile main.cpp dosyasını çalıştırıp katsayıları girmeniz yeterlidir. Sonuçlar virgülden sonra iki basamak hassasiyetle gösterilir.
